Summary

The Project is about the analyzing each players' strength using various measures and creating your Best Final 11 using recent T20 world cup (2022) data.

In this project I used the json source file and imported them in Python Pandas and then cleaned the data. After cleaning the data was the saved as .csv files for further analysis work. Various matrics and calculated columns where created to get the required result. The analysis of the cleaned data was done in Microsoft Power BI. Used buttons to navigate between different report pages. Used IF and ISFILTERED functions to display or hide the text & metrics as required
